## 1.数据库操作

1. 查看所有数据库

```sql
show databases;
```

2. 创建数据库

```sql
create database db;
```

3. 切换数据库

```sql
use db;
```

4. 查看当前使用的数据库

```sql
select database();
```

5. 删除数据库

```sql
drop database db;
```
